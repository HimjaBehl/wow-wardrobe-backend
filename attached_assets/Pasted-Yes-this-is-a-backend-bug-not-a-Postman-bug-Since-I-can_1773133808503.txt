## Fix 1: boolean flags being forced to `true`

### Search for this

```js

```

and:

```js

```

### So the final lines should look like

```js
const includeStaples =
  req.body.include_staples === undefined ? true : req.body.include_staples === true;

const includeWardrobe =
  req.body.include_wardrobe === undefined ? true : req.body.include_wardrobe === true;
```

### Why

Because this is wrong:

```js
const includeStaples = req.body.include_staples || true;
```

If Postman sends `false`, JavaScript turns that into `true`.
